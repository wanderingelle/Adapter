# Adapter
Emoji Adapter code for school assignment.

A simple C# code that shows how the Adapter Design Pattern works.

# What happens
The statement about minions is given and user is asked to express their emotions using :text emojis
The user types an emoji code like :fire and the program returns it to the matching emoji 🔥

# Available emojis
:fire  → 🔥
:lol   → 😂
:love  → ❤️
:agree → 👍
